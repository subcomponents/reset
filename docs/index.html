Abstract

Default styles for default elements. It will provide a presentable layer of elements on which to build upon.

Installation

CDN link:

<link rel="stylesheet" href="https://unpkg.com/hyper-reset-css@3.1.0/dist/reset.min.css"/>

NPM package:

$ npm install --save-dev hyper-reset-css

CSS import:

@import url("https://unpkg.com/hyper-reset-css@3.1.0/dist/reset.min.css");
