CSS reset library

Introduction

Default styles for default elements. It will provide a presentable layer of styles on which to build upon.

Installation

CDN link:

<link rel="stylesheet" href="https://unpkg.com/@subcomponents/reset@3.1.1/dist/reset.min.css"/>

NPM package:

$ npm install --save-dev @subcomponents/reset

CSS import:

@import url("https://unpkg.com/@subcomponents/reset@3.1.1/dist/reset.min.css");
